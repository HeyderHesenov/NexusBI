Hansı problemi həll edir?

Data var — cavab isə analitikin növbəsində gözləyir.

  • Darboğaz: hər sual üçün biznes komandası SQL bilən analitikə müraciət etməli olur.
  • Ləngimə: sadə bir “niyə düşdü?” sualı günlərlə gözləyir.
  • Alət dağınıqlığı: sorğu, qrafik, proqnoz, hesabat — hamısı ayrı yerlərdə.

NexusBI bunu self-serve edir: sualı verən şəxs cavabı, qrafiki və insight-ı dərhal özü alır.

Bölmə 01 — Analiz

Soruş, kəşf et, proqnozlaşdır.

Soruş/
Adi dildə sual; çox-turlu söhbət (davam sualları); SQL power-user rejimi + AI SQL-ini redaktə; TrustBadge; insight + stat-chip-lər; nəticəni dashboard-a saxla və ya qərara çevir.
Tarixçə/history
Bütün sorğuların jurnalı: axtar, qrafik tipinə görə filtrlə, bir kliklə yenidən işlət.
AutoML/automl
Cədvəl + hədəf seç → Linear/Logistic vs Random Forest avtomatik müqayisə; R²/Accuracy/F1, feature importance, CV, per-prediction “əsas drayverlər”.
Bölmə 02 — Məlumat

Mənbələr, məna qatı və keyfiyyət.

Mənbələr/sources
Postgres/MySQL/SQLite, CSV/Excel, Power BI. Explore (oflayn avto-dashboard), Schema, Profil, RLS, yerində Refresh, freshness SLA.
Metriklər/metrics
Biznes metriklərini bir dəfə təyin et; AI onları ardıcıl işlədir. Certify / verified nişanı.
Biliklər qrafı/graph
Asset-lərin interaktiv qrafı. Impact rejimi: seçilən node-un bütün downstream təsirləri.
Data müqavilələri/contracts
Not-null, Unique, Range, Schema, Freshness qaydaları. Yoxla → Pass/Fail.
Tələblər/requirements
BRD yüklə → ölçüləbilən KPI-ları çıxar → onlardan tam dashboard qur.
Bölmə 03 — Vizuallaşdırma

Canlı, paylaşıla bilən dashboard-lar.

Dashboard-lar/dashboards
AI ilə qur, Explore, widget sürüklə, Canlı rejim (real-time), Zaman maşını (snapshot), Hekayə, Komanda (canlı kursor + chat), Paylaş/Embed, qlobal filtr + cross-filter.
Hesabatlar/reports
Saxlanmış sorğular: yenidən işlət, avtomatik cədvəl (saatlıq/günlük/həftəlik), alert (hədd və ya z-score anomaliya), PDF/Excel çatdırılma.
Bölmə 04 — Planlama

İnsight-dan qərara və ssenariyə.

Qərarlar/decisions
Decision Loop: insight→hərəkət→nəticə. Baseline vs realized vs counterfactual; qərar dəqiqliyi.
Hədəflər/targets
KPI hədəfləri + pacing: attainment %, gözlənilən sürət (on-track / behind).
Digital Twin/twin
Metrik ağacı simulyatoru: Simulyasiya (±% levers), Goal-seek, Müqayisə, Monte Carlo (P10/median/P90).
BA Studio/ba-studio
SWOT, Porter 5 Qüvvə, BCG matris, Proses xəritəsi (BPMN) — AI + deterministik fallback.

Komanda, brend və planlar.

Komanda/workspaces
Workspace-lər + RBAC (owner/editor/viewer), audit log.
Brendinq/branding
Embed dashboard-lar üçün white-label (ad, rəng, loqo) + AA-kontrast yoxlaması.
Planlar/pricing
Abunə planları (Free/Pro/Max/Max+), aylıq AI-sorğu kvotası, istifadə.
Bildirişlər/notifications
Bildiriş mərkəzi + “Səhər brifi” — dəyişiklikləri bir icmalda toplayır.
İstənilən nəticə üzərində

Analiz panelləri.

Hər sorğu nəticəsinə tək kliklə dərinləşən analizlər qoşulur — ayrı səhifə yox.

Qrafik & Pivot
8 qrafik tipi + CSV, zoom, drill-down. Client-side cross-tab pivot explorer.
Proqnoz & Ssenari
Forecast (etibar intervalı) + goal-seek, Monte Carlo, what-if % düzəlişi.
Diaqnostika
Anomaliyalar (IsolationForest), Səbəb analizi (Pearson + BH-FDR), Etibarlılıq yoxlaması, Lineage.
“Niyə?”
Çoxsəviyyəli root-cause ağacı — drayverlər + töhfə %; AI oflayn olanda deterministik fallback.
Təhlükəsizlik & etibar

Fail-closed dizayn.

  • Yalnız-SELECT SQL guard — DML/DDL/çox-ifadəli sorğular DB-yə çatmadan rədd olunur.
  • SQL-səviyyə RLS — aqreqasiyadan əvvəl tətbiq olunur, cəm/qrup total-larını sızdırmır.
  • User-scoped cache — nəticə keşi istifadəçiyə bağlıdır, sızma yoxdur.
  • Şifrələnmə & rotasiya — Fernet ilə sirlər, SSRF guard, refresh-token reuse aşkarı.
  • Provenance — TrustBadge (əminlik/mənşə), metrik sertifikasiyası, lineage, freshness SLA.
